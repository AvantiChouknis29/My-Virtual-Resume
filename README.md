# My Portfolio
My Virtual Portfolio
This is my personal portfolio website built using the MERN (MongoDB, Express.js, React.js, Node.js) stack. It serves as a professional platform to showcase my technical skills, projects, work experience, education, and contact information. The application is designed with a clean and modern UI and optimized for responsiveness across all devices.

🧩 Key Features
Responsive and clean user interface

Projects, experience, and education sections powered by MongoDB

Contact form with backend integration

Seamless navigation and smooth scrolling

Real-time rendering of data from the backend

Deployed frontend via Netlify
